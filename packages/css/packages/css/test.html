Colors

  • --c[-(L|D)]-(P|A[|C])[-D[-I]]
  • --bc[-(L|D)]-(P|A[|C])

Legend:

  • L = light theme
  • D = dark theme
  • I = inverted
  • D = disabled
  • P = primary color swatch
  • A = accent color swatch
  • C = complement color swatch (optionally included)

Examples

  • --c-P - the primary color (whether it's the value from light or dark depends on the prefers-color-scheme value
  • --c-L-P-D-I - the disabled text color when using the inverted light primary theme
  • --bc-D-A - the background-color of the dark accent theme
